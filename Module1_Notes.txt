Módulo 01 - Git e versionamento de código | Module 01 - Git and code versioning

Status file on GIT | Estados do arquivo no GIT:

# Untracked | Não rastreado:
New files or deleted files | Arquivos novos ou excluídos

# Unmodified | Não modificado - Arquivos não modificados, não foram realizadas alterações | Unmodified files, no changes were made

# Modified | Modificados - Arquivos modificados | Modified files

# Staged - Arquivos prontos para commit, após estarem modificados | Files ready for commit, after being modified

COMMANDS  | COMANDOS - Git Bash

git status - Serve para verificar o estado de versão daquele arquivo | Serves to check the version status of that file

git add - Usado para adicionar os arquivos modificados ao próximo commit | Used to add the modified files to next commit

git diff - Usado para visualizar o que foi modificado no arquivo | Used to view what has been modified in the file

git commit -m - Used to add comments to commit | Usado para adicionar comentários ao commit

git log - Used to view the commits history | Usado para visualizar o histórico de commits

git restore - Restore the file for the last modified | Restaurar o arquivo para a última modificação

git fetch - View all modifications on the origin repository  | Ver todas as modificações no repositório que serão trazidas através do git pull

git branch - study later / estudar depois de forma mais profunda

git merge - Une as alterações de 2 branches, por exemplo
